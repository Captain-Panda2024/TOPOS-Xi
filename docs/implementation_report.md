## 4. コーディング規約と注意点

### 4.1 命名規則
- クラス名: PascalCase
- メソッド名: camelCase
- 変数名: snake_case
- 定数: UPPER_CASE

### 4.2 コードスタイル
- インデント: 4スペース
- 行の最大長: 80文字
- ブレース位置: 同じ行

### 4.3 ドキュメント規約
- すべてのパブリックAPIにドキュメントコメント
- 複雑なアルゴリズムには実装コメント
- 数学的な概念には参考文献を記載
